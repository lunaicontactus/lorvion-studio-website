데이터 삭제

LUNAI 계정과 연결된 서비스 데이터의 삭제 방법을 안내합니다.

앱에서 직접 삭제하기(권장)

  1. LUNAI 앱을 실행합니다.
  2. 마이페이지 → 계정 관리로 이동합니다.
  3. 계정 삭제를 선택합니다.
  4. 삭제되는 데이터 안내를 확인하고, 확인 문구를 입력합니다.
  5. 이메일로 계정을 보호 중이라면 비밀번호로 한 번 더 본인 확인을 합니다(익명 계정은 생략).
  6. 삭제가 완료되면 자동으로 로그아웃되고 시작 화면으로 이동합니다.

앱에 접근할 수 없는 경우

가입에 사용한 이메일 주소로 아래 문의처에 삭제 요청을 보내주세요. 확인 후 서버에서 동일하게 처리해 드립니다.

lorvionstudio@gmail.com

삭제되는 것

  • 계정 정보(닉네임, 로그인 정보)
  • 내가 친구방에 공유한 노래(음원·표지 파일 포함)
  • 내가 남긴 리액션
  • 오늘 감정 체크인 서버 기록
  • 기기에 저장된 감정 일기·앨범 등 로컬 기록(앱 내부 삭제 시)

다른 사용자의 대화를 지키기 위해 남는 것

친구방에 남긴 텍스트 채팅은 삭제되지 않고, 보낸 사람 표시만 "탈퇴한 사용자"로 바뀝니다(내용은 유지됩니다). 방장이었던 방은 다른 멤버가 있으면 방장 권한이 자동으로 넘어가고, 혼자였던 방은 함께 삭제됩니다.
